## Local Install
**Must use Node 16**

```
npm i
```


## Build TS Smart Contract to WASM+ABI
```
npm run build:eosio.token
```

## Run tests
```
npm run test:eosio.token
```

## Built using
Uses SDK: https://github.com/uuosio/ascdk
