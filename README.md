## Local Install
**Must use Node 16**

```
npm i
```


## Build TS Smart Contract to WASM+ABI
```
npm run build:token
```

## Run tests
```
npm run test:token
```

## Upload WASM+ABI to Blockchain
https://www.protonscan.io/wallet/utilities/upload-contract
